# Personal Portfolio

A responsive portfolio website built with plain HTML, CSS, and JavaScript.

## Open in VS Code

1. Unzip the project if needed.
2. Open the `personal-portfolio` folder in VS Code.
3. Open `index.html`.

For live previews, install the VS Code **Live Server** extension, then right-click
`index.html` and choose **Open with Live Server**.

## Main files

- `index.html` — page content and links
- `styles.css` — colors, typography, layout, and responsive styling
- `script.js` — mobile navigation and active-section behavior
- `project.html` — reusable long-form project case-study layout
- `project.css` — case-study styling and responsive layouts
- `project.js` — editable content, colors, and links for all project pages

Replace placeholder content such as `Your Name`, project descriptions, email,
and social links before publishing. To edit the three case studies, update the
project entries at the top of `project.js`.
